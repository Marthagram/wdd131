Farmers Hub in Niger State, Nigeria.

This name encompasses and group all the farmers farming in Niger State togther.

Site Purpose

This site supports farmers in this region to function as a diverse but one body by:

  • providing a membership form, in which farmers can apply to be members
  • providing complaint and suggestion box to the public, involving agriculture, these comments are relayed to the government and thoroughly looked into.
  • creating a table that contains physical platforms and areas across the state, where farmers can meet, learn from each other and talk about their concerns in farming and it can be solved.
